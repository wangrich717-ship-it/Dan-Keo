# Dan Koe Knowledge Base

85 essays by Dan Koe on business, writing, productivity, and the future of work.

**Features**
- Full-text search
- Category filtering
- Article reader with reading progress
- Mind map view for each article
- Bilingual (EN/中文) support

**Local development**
```bash
python build_data.py   # rebuild article data
python translate.py    # generate Chinese translations
python -m http.server 8080
```
